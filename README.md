# Emergency Supply Delivery Drone using KK 2.1.5

## Problem Statement
In small-scale environments, delivering essential items quickly can be difficult.

For example:
- Delivering medicines in hostels or campuses
- Transporting small tools across short distances

---

## Proposed Solution
This project implements a quadcopter drone capable of carrying small payloads for short-range delivery.

The drone provides:
- Stable manual flight
- Ability to carry lightweight items
- Faster delivery compared to manual transport

---

## System Overview
The drone uses a KK 2.1.5 flight controller for stabilization.

Working principle:
- Sensors detect orientation
- Controller adjusts motor speeds
- Drone maintains balance during flight

## Payload Capability
The drone is capable of carrying small lightweight objects such as:
- Medicine packets
- Small tools

---

## Key Learnings
- Effect of added weight on stability
- Importance of balance and thrust distribution
- Challenges in maintaining controlled flight with payload

## Limitations

- Limited payload capacity
- Short-range operation
